#### 5. ES 통합 검색 & Redis Stream (데이터 엔지니어링 & 비동기 처리)

- **매력 포인트:** RDB의 한계를 극복하기 위한 검색 엔진 도입, 그리고 분산 환경에서의 **데이터 정합성(Consistency)** 고민. Redis Stream을 쓴 것도 최신 기술 트렌드를 잘 반영한 것입니다.
- **경력기술서 표현 예시:**
    - "RDB의 `LIKE` 검색 성능 한계를 극복하기 위해 Elasticsearch 기반 통합 검색 엔진 구축."
    - "데이터 파편화를 막기 위해 Redis Stream을 활용한 비동기 이벤트 처리 파이프라인 구축 및 Batch 작업을 통한 최종적 데이터 정합성(Eventual Consistency) 확보."
