<p align="right">박 규 민</p>
(주)유비 (2024.01 ~ 재직 중)

- 직위: 연구원
- 역할: Legacy CMS 솔루션의 아키텍처를 현대화하는 마이그레이션 프로젝트를 주도하며, 인증,인가, 조직도 등 핵심 도메인의 설계를 담당합니다. 또한, 폐쇄망 환경에서의 데이터 파이프라인 구축 및 트레픽 처리를 위한 성능 최적화 업무를 수행합니다.
- 주요 책임: 
  1. Legacy CMS 솔루션의 MSA 전환 및 모던 아키텍처 마이그레이션(25.04 ~ 현재)
  2. 조직도 및 연락처 관리 대시보드 웹 애플리케이션 Full-stack 개발
  3. 폐쇄망-외부망 간 보안 데이터 연동 및 ETL 파이프라인 구축
- 주목할만한 성과:
1. 아키텍처 고도화 및 운영 안정성 확보
	- Jenkins 기반의 무중단 배포 파이프라인을 구축하여 배포 시 서비스 중단 시간을 제거하고 운영 안정성을 대폭 향상시켰습니다.
	- JWT와 Redis를 결합한 하이브리드 인증 아키텍처를 설계하여, 인증 검증 오버헤드를 줄이고 장애 발생 시에도 서비스 가용성을 유지하도록 개선했습니다.
	- 도메인 특성에 맞춰 Persistence Context(JPA, QueryDSL, JDBC)를 최적화하여 적용했습니다. 복잡한 조회는 QueryDSL로, 대량 데이터 처리는 JDBC로 이원화하여 유지보수성과 성능을 동시에 확보했습니다.
2. 웹 성능 최적화 및 사용자 경험 개선
	- SSR과 CSR의 하이브리드 렌더링 및 2단계 캐싱 전략(Spring Cache + Static JSON)을 도입하여, 초기 로딩 속도를 개선하고 Lighthouse 성능 점수 93점(상위 5%)을 달성했습니다.
	- 대규모 조직 데이터를 처리하기 위해 Recursive Tree 구조를 클로저(Closure)로 캐싱하고, Lazy Loading과 History API(pushState)를 활용하여 깊은 계층의 데이터도 지연 없이 렌더링되도록 최적화했습니다.
3. 데이터 처리 시스템 및 기능 확장
	- Elasticsearch 기반의 통합 검색 엔진을 개발하고, Redis Stream을 활용한 비동기 이벤트 처리로 데이터 색인 속도와 검색 정확도를 향상 시켰습니다.
	- 런타임 동적 스키마 제어 엔진 개발: 클라이언트 요청에 따라 테이블 생성/수정/삭제가 가능한 게시판 관리 시스템을 구축하여, 서비스 확장성을 극대화하고 관리자 편의성을 100% 개선했습니다.
4. 이기종 데이터 동기화 및 ETL 자동화
	- 폐쇄망 환경에서 7만 건 이상의 XML 데이터를 수집/정제하여 외부망으로 전송하는 End-to-End 데이터 파이프라인을 구축했습니다.
	- 파일 기반 커서 및 인메모리 중복 제거 로직을 자체 구현하여 데이터 정합성을 확보하고, Java-Python 간 소켓 통신 최적화를 통해 네트워크 대역폭 비용을 약 80% 절감했습니다.
